**EchoSend** is an API testing tool. It is similar to Postman.
EchoSend is seamlessly integrated into EchoProxy, and brings much more convenience for users when resending requests.

> **Entrance**: Click **left sidebar** button <TablerSend style={inlineIcon} />  

<Image src={sendImage} style="border-radius:6px;" alt="EchoSend" />

## Resend
You can resend any captured request by **right click** on the record,
and click <TablerSend style={inlineIcon} /> **EchoSend**.  
All HTTP request fields will be **automatically** filled in EchoSend panel.
